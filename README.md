# dev
1. Duplicate file .env.template a .env
2. Configure environment variables
```
PORT=3000

MAILER_EMAIL=
MAILER_SECRET_KEY=

PROD=false

MONGO_URL=
MONGO_DB_NAME=
MONGO_USER=
MONGO_PASS=

```
3. Ejecute command ```npm install```
4. Ejecute command ```npm run dev```
